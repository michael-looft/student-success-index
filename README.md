# Student Success Index

A predictive analytics tool for identifying students at risk of academic difficulty during the 8th-to-9th grade transition, using Weight of Evidence (WOE) and Information Value (IV) methodology.

Built to help school counselors prioritize outreach before and during the critical first six weeks of high school.

> **Note:** All students, schools, and data in this repository are entirely synthetic and fictional. The methodology was developed using real district data; this version uses fabricated records for public demonstration purposes.

## The Problem

The transition from middle school to high school is one of the highest-risk periods in a student's academic career. Students who receive a D, F, or Incomplete (DFI) in their first semester of 9th grade are significantly more likely to fall off track toward graduation. Counselors often have limited time and need a way to prioritize which incoming students to connect with first.

## The Approach

The Student Success Index (SSI) scores each incoming 9th grader on a 0-100 scale based on their 8th grade academic record, placing them into one of three risk categories:

| Category | Score Range | Counselor Action |
|---|---|---|
| **High Risk** | 0-39 | Priority outreach before school starts; Summer Bridge referral |
| **Monitor** | 40-64 | Check-in during first 6 weeks; targeted support |
| **On Track** | 65-100 | Continue supporting strengths |

## Validation (Synthetic Data)

On the 180-student synthetic dataset, the SSI correctly differentiates risk:

| Risk Category | Students | Actual DFI Rate |
|---|---|---|
| High Risk | 83 | **69%** |
| Monitor | 60 | 38% |
| On Track | 37 | **19%** |

High Risk students experienced DFI at 3.6x the rate of On Track students.

## Key Design Decision — Controllable vs. Non-Controllable Factors

The scoring **algorithm** includes both:
- **Controllable factors:** GPA (overall, math, science, ELA), attendance
- **Non-controllable factors:** race/ethnicity, English learner status, economic disadvantage, special education status

Both types genuinely predict outcomes, and omitting non-controllable factors would reduce accuracy. However, the counselor-facing **report card** shows only controllable factors as strengths and challenges. Telling a student that their race or household income is a "challenge" is deflating and unhelpful — they cannot change those things. Counselors see the full score; students and families see only actionable insights.

This mirrors standard practice in medical risk modeling: a physician's full risk calculation may include age and family history, but the patient's care plan focuses on diet, exercise, and medication.

## Methodology

### Weight of Evidence (WOE)

WOE measures how strongly each value of a predictor is associated with the outcome (9th grade DFI). For each bin or category:

```
WOE = ln(Distribution of DFI Students / Distribution of Non-DFI Students)
```

Positive WOE → associated with higher risk  
Negative WOE → associated with lower risk

### Information Value (IV)

IV summarizes the overall predictive power of each variable:

```
IV = Σ (Distribution of DFI - Distribution of Non-DFI) × WOE
```

| IV Range | Predictive Strength |
|---|---|
| < 0.02 | Unpredictive |
| 0.02 - 0.10 | Weak |
| 0.10 - 0.30 | Medium |
| > 0.30 | Strong |

### SSI Score Construction

```
Raw Score = Σ (WOE × IV weight) for each feature
SSI Score = Normalized to 0-100 (higher = better prepared)
```

## Usage

```bash
# Install dependencies
pip install pandas numpy

# Step 1: Generate synthetic student data
python generate_synthetic_data.py

# Step 2: Run the SSI model
python student_success_index.py

# Use your own data file
python student_success_index.py --input your_students.csv
```

### Input File Format

Your CSV should include these columns (names can be adjusted in the script):

| Column | Description |
|---|---|
| `grade8_overall_gpa` | 8th grade cumulative GPA (0.0-4.0) |
| `grade8_math_gpa` | 8th grade Math GPA |
| `grade8_science_gpa` | 8th grade Science GPA |
| `grade8_ela_gpa` | 8th grade ELA GPA |
| `grade8_attendance_pct` | Attendance percentage (0-100) |
| `race_ethnicity` | Race/ethnicity category |
| `english_learner` | Yes/No |
| `economically_disadvantaged` | Yes/No |
| `special_education` | Yes/No |
| `gender` | Gender identity |
| `grade9_any_dfi` | Outcome: 1 if student received any D/F/I in 9th grade |

## Context

This tool was developed at [Marin Promise Partnership](https://www.marinpromisepartnership.org) in collaboration with a data science partner, as part of a broader effort to use predictive analytics to support student success in the 8th-to-9th grade transition. The methodology is adapted from credit risk scoring (WOE/IV is widely used in financial modeling) and translated into an education equity context.
